- Tool Overview

Category
File Share/Transfer
Description
Connects to shared folders that are publicly available on the network.
Example of Presumed Tool Use During an Attack
This tool is used to send in tools to be used for attacks via shared folders and to acquire information from a file server.

- Tool Operation Overview

Item Source host Destination Host
OS Windows
Belonging to Domain Not required
Rights Standard user
Communication Protocol 445/tcp
Service Workstation Server

- Information Acquired from Log

Standard Settings
  • Source host
    • Execution history (Prefetch)
Additional Settings
  • Source host
    • Execution history (audit policy, Sysmon)
    • Communication using 445/tcp (audit policy, Sysmon)
  • Destination Host
    • Communication using 445/tcp (audit policy, Sysmon)
    • Access history to files and access to a share path are recorded. In addition, when auditing of a file system is enabled, access history to a target file is recorded (audit policy)

- Evidence That Can Be Confirmed When Execution is Successful

  • Source Host: The Event ID 4689 (A process has exited) of net.exe was recorded in the event log "Security" with the execution result (return value) of "0x0".
  • Destination host: In the Event ID 5145 of the event log "Security", the fact that files were read is recorded.
