- Tool Overview

Category
Deleting Evidence
Description
Deletes Windows event logs.
Example of Presumed Tool Use During an Attack
This tool is used to delete the evidence of an attack.

- Tool Operation Overview

Item Source Host Destination Host
OS Windows
Belonging to Domain Not required
Rights Standard user Administrator
Communication Protocol 135/tcp

- Information Acquired from Log

Standard Settings
  • Source host
    • Execution history (Prefetch)
  • Destination Host
    • A record that an event log was cleared (each event log)
Additional Settings
  • Source host
    • Execution history (audit policy, Sysmon)
    • The account used for clearing logs and the host that executed the clear command (Sysmon).

- Evidence That Can Be Confirmed When Execution is Successful

  • The Event ID: 104 indicating that a log was cleared is recorded at the beginning of the event log.
