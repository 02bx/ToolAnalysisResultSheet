- Tool Overview

Category
Password and Hash Dump
Description
Extracts user names and passwords saved in the web browser.
Example of Presumed Tool Use During an Attack
This tool is used to extract and use account information entered for accessing an intranet or external services.

- Tool Operation Overview

Item Description
OS Windows
Belonging to Domain Not required
Rights Standard user

- Evidence That Can Be Confirmed When Execution is Successful

  • The successful execution of the tool cannot be determined from event logs or execution history. Note: If the extracted password is saved, it is considered that the execution was successful.
